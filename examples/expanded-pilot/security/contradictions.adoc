Manually-authored cross-references linking conflicting claims. V5 does not

::contradiction auth.session.conflict
severity: high
status: unresolved
claims: [auth.session.local-storage-allowed, auth.session.memory-storage, auth.session.csrf-protection]
--
Claim auth.session.memory-storage requires memory-only storage while
auth.session.local-storage-allowed permits localStorage. This is an unresolved
conflict in session-storage guidance. The auth.session.csrf-protection claim is
also under review for interaction effects with session-storage choices.
::
